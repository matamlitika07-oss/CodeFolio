## FIX 2: Profile completion calculation
Calculate profile completion percentage dynamically:

const calculateCompletion = (userData) => {
  const checks = [
    userData?.profile?.name,
    userData?.profile?.bio,
    userData?.profile?.avatarUrl,
    userData?.profile?.socialLinks?.github,
    userData?.profile?.socialLinks?.linkedin,
    userData?.projects?.length > 0,
    userData?.skills?.frontend?.length > 0,
    userData?.templateId && userData?.templateId !== 'minimalist'
  ];
  const filled = checks.filter(Boolean).length;
  return Math.round((filled / checks.length) * 100);
};

Show this percentage in the sidebar and animate the progress bar width with CSS transition.
